Використання програми:
При методі шифрування Цезарь програма може зашифрувати текст безмежної довжини.
Програма шифрує файли .txt та .py
Програма не шифрує великі літери, для шифрування їх потрібно перевести в нижній регістр вибравши пункт: Remove capital letters
Для дешифрування файла з зашифрованим текстом, в ньому обов'язково моє бути ключ.

При методі шифрування RSA програма може зашифрувати текст довжиною відповідно до відношеня: 2048 біт = 128 символів, тобто 1 символ = 16 біт.
Мінімально можлива довжина ключа це 2048, максимальна довжина ключа не обмежена програмою, але рекомендую використовувати максиум 4096.
Програма шифрує тільки файли .txt
